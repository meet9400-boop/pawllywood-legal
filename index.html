Pawllywood Uploader

A personal content-automation tool for the Pawllywood short-form video channel.

What it does

Pawllywood Uploader is a private tool used by the owner of the Pawllywood channels to publish the owner's own original short-form videos to the owner's own accounts on TikTok, Instagram, and YouTube through each platform's official API.

How it works

  • The operator connects their own TikTok account via TikTok's official Login Kit (OAuth).
  • The operator selects a finished video and its caption.
  • The tool uploads the video to the operator's own account using the TikTok Content Posting API (video upload & publish).
  • The tool posts only the operator's own content; it does not access or interact with any other user's account or data.

Content disclosure

Videos may contain AI-generated imagery and are labeled using each platform's AI-content disclosure features.
